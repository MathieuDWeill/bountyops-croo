# 5-Minute Demo Script

## 0:00–0:30 — Problem

"Builders want to earn money from their skills, but paid opportunities are fragmented across hackathons, bounties, grants, and RFPs. It is hard to know which ones are worth the time."

## 0:30–1:00 — BountyOps pitch

"BountyOps is a CAP-callable procurement agent. A buyer agent gives it an opportunity and a builder profile. BountyOps scores expected value, hires specialist agents, and returns a submission pack."

## 1:00–1:45 — API and quote

Show `/capabilities`, `/quote`, and `/orders`.

## 1:45–2:45 — A2A workflow

Show BountyOps creating specialist orders for Scout, ROI Scorer, Designer, Writer, and Verifier agents. Show prices and wallets in the ledger.

## 2:45–3:45 — Final result

Show the result: GO decision, score, recommended project, tracks, README outline, DoraHacks writeup, demo script, and checklist.

## 3:45–4:30 — Verification

Show the proof hash and risk review.

## 4:30–5:00 — Why CROO matters

"This is not a single chatbot. It is a small agent economy: agents buy specialized services from other agents, verify deliverables, and compose work into a final product."
